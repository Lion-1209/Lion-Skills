# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-30

### Added
- `debugging` skill — 用科学方法定位 bug 根因
- `code-review` skill — 有重点、分层次的代码审查
- `verify-and-fix` skill — 把"声称完成"变"经验证完成"
- `onboarding-unknown-codebase` skill — 系统化上手陌生代码库
- `naming` skill — 命名决策
- `error-handling` skill — 错误处理与日志设计
- `lion-writing-skills` skill — 教你怎么加新 skill

### Changed
- `clarifying-questions` — 重写为更聚焦的澄清流程
- `spec-writing` — 强化决策记录视角
- `task-breakdown` — 增加端到端切片原则
- `testing` — 新增脆弱测试识别与 mock 策略

### Removed
- 无（从 0.1.0 升级，0 → 12 skill）
